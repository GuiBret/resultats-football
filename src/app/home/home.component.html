Bonjour,


Vous êtes actuellement sur mon application de résultats / classements de football.

Les données de celle-ci sont extraites de l'API football-data, puis sont récupérées et traitées via un backend Node.js dont le repo se trouve ici.

Dans les pages des compétitions récupérées, vous pouvez voir le classement, les résultats et les équipes inscrites dans celles-ci.

Les compétitions sont accessibles de 2 façons : soit sous la forme d'une liste les affichant toutes, soit sous la forme d'une recherche par pays.
